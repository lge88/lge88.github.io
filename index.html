I am a PhD student in Department of Structural Engineering at University of Califonia, San Diego. My advisor is Professor Falko Kuester.

My research interests are simulation-based engineering, computational mechanics, computer graphics, cloud computering and human-computer interaction.

I am a big fan of open source softwares. I have enjoyed a considerable boost to my research from many open source projects, and I also try to make the libraries and toolkits that I created available and reusable to others.

Projects

  • ARStructure 3D structure modeling and simulation system in augumented reality environment. ARStructure is a mobile application that allows user to build, simulate and visualize structure model on top of the real camera scene. The constructed model, as well as the simulation results, are augumented and overlaped with the real object, presenting an intuitive and interesting aspect to understand the structure behaviour.
  • SketchIT A Sketch-based modeling and simulation enviroment for structural analysis. SketchIT allows user easily and quickly construct a conceptual structure model by sketching. The constructed model can be simulated and visualized on the fly. The simulation runs on the remote server using FEM (Finite Element Method). Video demos can be found here: staic analysis , dynamic analysis.
  • IFEA Integrated Finite Element Analysis web framework. IFEA is the foundation upon which SketchIT and ARStructure are built. IFEA offers services to common tasks in FEA process such as 3D modeling, mesh generation, simulation, result visualization and data sharing via a simple JSON-based API. The services are mainly implemented in Node.js, a high performance event-driven server-side software system. IFEA also provides a collection of reusable web components (HTML, css, Javascript) that take the advantage of latest web technology (HTML5, WebGL, WebSocket) to build rich web interface.
  • node-opensees Node.js binding for OpenSees (Open System for Earthquake Engineering Simulation). The default FEA solver for IFEA.
  • node-gmsh Node.js binding for Gmsh, an open source three-dimensional finite element mesh generator. The default mesh generator for IFEA.
